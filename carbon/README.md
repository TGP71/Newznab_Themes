Newznab-Carbon-Theme
====================

Dark theme for newznab plus.

To install:

git clone https://github.com/kevinlekiller/Newznab-Carbon-Theme

cd Newznab-Carbon-Theme

sudo mv carbon /path/to/newznab/www/templates
